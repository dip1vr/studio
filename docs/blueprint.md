# **App Name**: Vidyarthi Vista

## Core Features:

- AI Provider Configuration: Users can enter and select API keys for various AI providers (Google Gemini, Anthropic Claude, OpenAI ChatGPT, DeepSeek) via a settings page. Keys are stored locally for dynamic AI interaction.
- Customizable Exam Setup: A step-by-step wizard to select exam type, optional subjects/topics, number of questions, difficulty, time limit, negative marking rules, and language preference (Hindi/English/Bilingual).
- AI-Powered Question Generation: The selected AI provider will generate MCQ questions based on the user's chosen exam, subject, topic, and difficulty settings, creating a dynamic test paper.
- Interactive Test Interface: Displays questions one-at-a-time or all at once, with MCQ options, a clear timer, question palette for navigation, and actions like Mark & Review, Skip, or Clear Response.
- AI-Driven Report & Review: After test submission, a detailed report shows score, accuracy, section-wise breakdown, and AI-generated explanations for each question. The AI will also generate 3-5 personalized improvement tips.
- Local Test History & Progress: Stores past test results locally (date, exam, score, accuracy) and provides a simple graph to track performance and improvement over time.
- Bilingual & Responsive UI: Supports Hindi and English languages throughout the app, offering a clean, modern, and mobile-responsive interface with dark/light mode toggle for optimal readability.

## Style Guidelines:

- Color Scheme: A light color scheme emphasizes clarity and professionalism, suitable for an educational application and prolonged screen time during mock tests. This choice promotes focus and reduces eye strain.
- Primary Color: A thoughtful, subdued blue-violet hue (#5D5C8A). This color conveys a sense of depth, stability, and trust, aligning with the seriousness and educational nature of government exams. Its versatility allows for various shades without being distracting.
- Background Color: A very light, desaturated blue-violet (#F7F7F9). This background softly complements the primary color while ensuring maximum readability for extensive text content and maintaining a clean, open feel, consistent with modern exam portals.
- Accent Color: A bright, yet grounded blue (#4C86BC). Positioned analogously on the color wheel to the primary, this accent provides a clear contrast without clashing, ideal for highlighting interactive elements like buttons, selections, and important information, guiding the user's attention effectively.
- Headlines and Body Text Font: 'Inter', a grotesque-style sans-serif. Chosen for its modern, highly legible, and objective aesthetic, it provides excellent readability for both question text and detailed reports, ensuring a professional and accessible experience across all devices.
- Use clear, intuitive, and modern iconography with a minimalist design to enhance user navigation and interaction without adding visual clutter. Icons should be easily understandable in both dark and light modes.
- Employ a clean, structured, and adaptive layout, prioritizing readability and ease of navigation. The design will ensure consistent information hierarchy, accommodating a multi-step exam setup, detailed test interface, and comprehensive reports efficiently on various screen sizes.
- Implement subtle and purposeful animations for state changes (e.g., question navigation, selection highlights, loading states) to provide smooth user feedback and enhance perceived responsiveness without being distracting during critical tasks like taking a test.
